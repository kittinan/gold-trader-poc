# Frontend Application

Gold Trader Frontend Application

## Technology Stack
- React.js
- WebSocket Client
- Axios
- Styled Components
- Chart.js (for price visualization)

## Development
This directory contains the frontend React application for the Gold Trading platform.

Key responsibilities:
- UI/UX Design (Art)
- API Integration (Thaba)
- Real-time data display
- User interface components
- Trading interface

## Structure
```
frontend/
├── src/
│   ├── components/    # React components
│   ├── pages/         # Page components
│   ├── services/      # API services
│   ├── utils/         # Utility functions
│   └── styles/        # Styling files
├── public/            # Static files
└── package.json       # Dependencies and scripts
```
